# Avance-Proyecto---Progreso-2
Este repositorio es para el correcto almacenamiento de los commits de todos integrantes de grupo para el avance del proyecto

## Descripción General

Este proyecto tiene como objetivo desarrollar una solución computacional en modo gráfico, funcional e interactiva, aplicando estructuras de datos y trabajo colaborativo en equipo.

El avance del proyecto contempla la creación de al menos dos módulos gráficos completos, el uso de estructuras de datos como listas, colas, pilas u otras, la documentación del código fuente y la presentación del funcionamiento mediante una demostración grupal.

## Funcionamiento General del Sistema

El usuario interactúa con la aplicación mediante una interfaz gráfica. Cada módulo permite realizar acciones específicas relacionadas con la solución del problema planteado.

El flujo general del sistema es el siguiente:

1. El usuario accede a la interfaz principal.
2. Selecciona el módulo que desea utilizar.
3. Ingresa o selecciona información mediante los componentes gráficos.
4. El sistema procesa los datos usando estructuras de datos.
5. Se muestran los resultados en pantalla.
6. El usuario puede continuar interactuando con otros módulos.
